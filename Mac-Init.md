# 💻 New Mac Setup Manifest (2026)

A high-performance, AI-ready DevOps and Development environment.

## 🐚 Terminal & Shell Environment
* **Homebrew**: The foundation for all package management on macOS.
* **Oh My Zsh**: Framework for managing Zsh configuration and themes.
* **Zsh Plugins**:
    * `zsh-syntax-highlighting`: Fish-like syntax highlighting for the shell.
    * `zsh-autosuggestions`: History-based command completions.
* **fzf**: A command-line fuzzy finder for files and history.
* **Starship**: A fast, customizable Rust-based shell prompt.

## 🛠 Modern CLI Tools (Rust-based)
* **Rustup**: The Rust toolchain installer (Rustc, Cargo).
* **Alacritty**: GPU-accelerated terminal emulator (configured with Gatekeeper bypass).
* **Zellij**: A modern terminal workspace manager and multiplexer.
* **zoxide**: A smarter `cd` command that learns your habits.
* **eza**: A feature-rich replacement for `ls` (aliased to `ll`).
* **bat**: A `cat` clone with syntax highlighting and Git integration.
* **ripgrep (rg)**: Extremely fast recursive search tool.
* **fd**: A simple, user-friendly alternative to `find`.
* **bottom (btm)**: A graphical, real-time system/process monitor.
* **jq**: Command-line JSON processor for API data.

## 🏗 DevOps & Infrastructure
* **OrbStack**: Lightning-fast, lightweight alternative to Docker Desktop and Kubernetes.
* **Terraform**: Infrastructure as Code (installed via HashiCorp tap).
* **tflint**: A linter for identifying Terraform errors and best practices.
* **tfsec**: Static analysis security scanner for Terraform templates.
* **Ansible**: Automation and configuration management engine.

## ✍️ Development & Productivity
* **Visual Studio Code**: Primary code editor (installed via Homebrew Cask).
* **Maccy**: Lightweight, keyboard-first clipboard manager.
* **BreakTimer**: Open-source reminder tool to manage eye strain and the 20-20-20 rule.
